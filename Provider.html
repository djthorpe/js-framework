Class: Provider

Provider(constructor, origin)

Provider requests data from a remote endpoint.

Constructor

new Provider(constructor, origin)

Create a provider, with a model constructor and optionally a base URL for requests. If no model is provided then Object types will be emitted. Add event listeners for mvc.provider.{started,completed,added,changed,deleted} to tap into the request lifecycle and mvc.provider.error to deal with request errors.
Parameters:
Name Type Description
constructor Model The model used to create objects from data.
origin string The base URL used for making requests.
