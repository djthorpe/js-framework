Class: Provider

Provider(constructoropt, originopt)

The provider can store model objects which can be retrieved after a request is completed. If no model is provided then Object types will be emitted. Add event listeners to tap into the request lifecycle. To create a provider, with a model constructor and optionally a base URL for requests.
